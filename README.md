# Simulate AI Resource Optimization Batching

This example demonstrates how AI processing can consume system resources (CPU, memory) and how batching can optimize this. It compares an unoptimized approach, which processes all data at once, with an optimized approach that processes data in smaller batches, reducing peak resource demands. This helps prevent your system from being 'starved' by intensive AI tasks.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`.
3. Observe your system's Task Manager/Activity Monitor for CPU and Memory usage during execution.
